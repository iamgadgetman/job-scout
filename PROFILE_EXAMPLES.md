# Profile Configuration Examples

This document provides example candidate profiles for different roles and experience levels. Copy and modify these templates in your workflow's "Format Claude Prompt" node.

## Example 1: Senior DevOps/SRE Engineer

```javascript
CANDIDATE PROFILE — Alex Chen
- Location: Remote-only (US East Coast), willing to work West Coast hours, 0% travel
- Experience: 12 years in DevOps, SRE, and platform engineering
- Current Role: Staff Site Reliability Engineer at FinTech Startup (2021–present)
  • Architected multi-region Kubernetes platform serving 5M daily users
  • Reduced incident response time 70% through automated remediation
  • Led migration from EC2 to EKS, saving $400K annually
  • Mentored team of 8 SREs on observability best practices

- STRONG / EXPERT skills:
  • Container Orchestration: Kubernetes (CKA certified), Docker, Helm, ArgoCD
  • Cloud Platforms: AWS (Solutions Architect Pro), GCP (ACE), Terraform
  • Observability: Prometheus, Grafana, Datadog, New Relic, distributed tracing
  • Languages: Python (expert), Go (proficient), Bash (expert)
  • CI/CD: Jenkins, GitLab CI, GitHub Actions, Spinnaker
  • Incident Management: PagerDuty, on-call rotations, postmortem culture
  • Databases: PostgreSQL, MySQL, Redis, MongoDB (operations & tuning)

- LIMITED / DEVELOPING:
  • Rust: learning for performance-critical services
  • Machine Learning Ops: basic experience with model deployment
  • Azure: minimal hands-on experience

- Certifications: AWS Solutions Architect Pro, CKA, HashiCorp Terraform Associate
- Salary: $180k minimum (currently at $175k + equity)
- Hard dealbreakers: No Windows-heavy environments, no on-site requirements, no legacy mainframe

TARGET ROLES (good fit):
Staff/Principal SRE, Platform Engineer, DevOps Architect, Infrastructure Lead,
Cloud Architect, Reliability Engineer

NOT A FIT FOR:
Pure software development, frontend roles, data engineering, IT support,
traditional sysadmin, sales engineering

AUTOMATIC DISQUALIFIERS (score 20 or below):
- Requires on-site/hybrid (even 1 day/month)
- Windows Server or mainframe as primary platform
- Below $180k base salary
- Requires security clearance
```

## Customization Tips

### 1. Skills Organization
- List your strongest skills first
- Group by category (e.g., Cloud, Networking, Security)
- Be specific about proficiency levels
- Include version numbers for important tools

### 2. Salary Requirements
- Set a realistic minimum based on your market
- Consider total compensation if equity matters
- Account for regional differences

### 3. Dealbreakers
- Be honest about travel limits
- Specify work arrangement preferences clearly
- List technology you won't work with
- Include company size/stage preferences

### 4. Target Roles
- List 6-10 role titles that match your goals
- Include variations (Engineer/Developer/Architect)
- Consider industry-specific titles

### 5. Disqualifiers
- Set clear criteria for automatic low scores
- Include both hard requirements and strong preferences
- Consider which skills you absolutely won't use
